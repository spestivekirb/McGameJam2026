# Echo of Me - McGameJam 2026 
This is a fork of the repository used for our team's submission for the 48-hour McGameJam 2026 Hackathon in January 2026, at McGill University.
- The premise of the game is that your character is followed by a "ghost" that will kill on contact, but also interacts with the world as the player does. The ghost is capable of doing things such as pressing buttons.

## My Contributions
- My contributions were mainly to do with level design and coding game mechanics.
- **Level Design:** Designed some of the early levels and the final few levels (toggleable block levels).
- **Interactables:** Developed the scripts for button interactions with doors and toggable blocks. Also made unused mechanic that would speed up/slow down the ghost relative to the player.
- **Ghost "World":** Designed and implemented the logic for the "world" that the ghost is capable of interacting with, effectively a past copy of the player's.
- **Bug Fixes:** Resolved issues between Mac and Windows builds, as well as bugs revolving around the shadow mechanic.
